DHCP 服务安装和配置

在云计算机 A 中操作

安装

使用 sudo yum install dhcp -y 安装，使用 dhcpd 检查是否成功安装 dhcp

配置

修改 IP

  • 在 A 中使用 nmtui 命令工具修改 ens33 网卡 IP 为 192.168.24.1, 然后重新激活网卡

  • 在 B 中使用 nmtui 命令修改 ens33 网卡 IP 为自动分配

  • 在 C 中找到网络连接把本地连接 2 改成自动分配 IP
